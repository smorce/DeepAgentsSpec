# MiniRAG Demo UI (Static)

This service provides a static HTML/JS demo UI for MiniRAG.
It serves a minimal chat-like interface for register/search/delete operations
against the MiniRAG backend (api-gateway).

## Run locally

```bash
# From repository root
./services/frontend/EPIC-API-001-minirag-demo-ui/scripts/run_dev.sh
```

Then open <http://localhost:5173> in your browser.

See `service-architecture.md` for boundaries and `scripts/` for helper commands.
